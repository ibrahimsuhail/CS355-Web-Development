Get Started as a Student

Some Things to Note:

  1. There is a critical bug when you register an account as let's say a teacher, then log out and register a new student account and log in to that account.
    If you now try to log in the teacher account, it'll treat you as a student.
    So you must now create a new teacher account.
    This behavior happens Vice Versa.
  2. If you are a teacher and add a word-answer type question to a question set, it will always appear at the bottom for the student, but this won't mess up the score calculation.
